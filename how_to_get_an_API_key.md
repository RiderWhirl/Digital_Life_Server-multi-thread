# 如何获取API密钥

## ChatGPT
 - 访问 [OpenAI platform(platform.openai.com)](https://platform.openai.com/) 创建一个账户或登录。
 - 登录后，进入网站的API部分。通常可以在仪表板或特定的API选项卡下找到。
 - 按照说明创建新的API密钥。这通常包括选择订阅计划（免费或付费）并同意服务条款。
 - 创建完成后，系统会提供一个API密钥。请妥善保存该密钥。

## Deepseek
 - 访问 [Deepseek platform(platform.deepseek.com)](https://platform.deepseek.com/sign_up) 创建一个账户或登录。
 - 登录后，默认会送你10块钱的“赠送余额”（有效期为一个月），进入网站左侧的“API密钥”选项，点击“创建新的API密钥”按钮，按照提示生成密钥。
 - 创建完成后，系统会提供一个API密钥。请妥善保存该密钥。
